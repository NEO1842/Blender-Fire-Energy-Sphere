# 🤝 Contributing

## 📋 Workflow
1. Create an Issue
2. Set Priority (Triage → Critical → High → Medium → Low → Backlog)
3. Work on it
4. Submit a Pull Request

---

## 🐛 Bugs
- Describe clearly
- Add steps to reproduce

---

## ✨ Features
- Explain the idea
- Describe the benefit

---

## 🔧 Pull Requests
- Keep changes focused
- Link related issues

---

## 🎯 Rules
- Keep things simple
- Stay organized
